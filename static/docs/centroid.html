turf.centroid

Takes a Feature or FeatureCollection of any type and calculates the centroid using the geometric mean of all vertices. This lessens the effect of small islands and artifacts when calculating the centroid of a set of polygons.

Parameters

Name Type Description
fc FeatureCollection

a Feature or FeatureCollection of any type

Returns

Point

a Point feature at the centroid of the input feature(s)

Example

var poly = turf.polygon([[
	[105.818939,21.004714],
	[105.818939,21.061754],
	[105.890007,21.061754],
	[105.890007,21.004714],
	[105.818939,21.004714]
]]);

var centroidPt = turf.centroid(poly);

var result = turf.featurecollection([poly, centroidPt]);
